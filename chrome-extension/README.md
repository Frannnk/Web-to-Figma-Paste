# Web to Figma Paste (Chrome Extension Only)

This extension is **Chrome-only** and targets your requested flow:

1. Open webpage.
2. Click extension icon.
3. `Copy Full Page` or `Pick Element`.
4. Go to Figma canvas.
5. Press `Cmd/Ctrl + V`.

No local bridge service. No Figma plugin.

## Install

1. Open `chrome://extensions`
2. Enable Developer Mode
3. Load unpacked -> select `chrome-extension`

## Notes

- Pasting quality depends on target page complexity.
- If a specific page fails, use `Pick Element` for smaller scopes.
